--------------------------------------------------
4) DATA LAYER (NO BACKEND; IN-MEMORY)
--------------------------------------------------
File: lib/services/data_service.dart

This file acts as a dummy in-app database using static collections + ValueNotifier.

Main stores/notifiers:
- totalScoreNotifier: global student points
- currentUserNotifier: logged-in AppUser
- _usersDb: role+email keyed auth database
- postedCoursesNotifier: teacher-created courses (CoursePost list)
- studyMaterialsNotifier: List<Map<String, dynamic>> for uploaded materials
- teacherQuizNotifier: teacher quiz questions
- dailyGkPostsNotifier
- announcementsNotifier
- studentDoubtsNotifier
- _attemptsDb: tracks course/quiz attempts and marks for teacher analytics

Teacher results aggregation:
- getTeacherResultSummary(teacherEmail)
  returns students attempted course, attempted quiz, total marks, and per-student marks.
